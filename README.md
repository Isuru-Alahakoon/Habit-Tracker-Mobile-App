# 🌿 WellnessMate
### Personal Wellness & Daily Health Routine Manager (Android – Kotlin)

WellnessMate is an Android mobile application designed to help users build healthy routines, track emotions, and stay hydrated throughout the day.  
The app combines habit tracking, mood logging, and smart reminders into a simple and user-friendly experience.

This project was developed using **Android Studio** as part of an academic assignment to demonstrate Android development principles, UI/UX design, navigation, and data persistence **without using databases**.

---

## 📌 Overview

Maintaining personal wellness can be difficult in busy lifestyles. People often forget small but important habits like drinking water, exercising, or taking mental breaks.

**WellnessMate helps users:**

✔ Track daily habits  
✔ Record their mood  
✔ Receive hydration reminders  
✔ Monitor progress  

All within one lightweight and easy-to-use mobile app.

## ⭐ Core Features

### 📝 Daily Habit Tracker
- Add new habits (drink water, meditation, walk, etc.)
- Edit or delete habits  
- Mark habits as complete  
- View daily completion progress  

---

### 😊 Mood Journal with Emoji Selector
- Record mood with emoji  
- Automatically saves date & time  
- View history of past moods in a list  

---

### 💧 Hydration Reminder
- Set reminder interval  
- Notifications remind users to drink water  
- Built using Android notification tools *(AlarmManager / WorkManager)*

User data such as:

- Mood records  
- Reminder intervals  
- Preferences  

are stored using **SharedPreferences**.
